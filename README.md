# Diabetes Detection Using Machine Learning

This project applies supervised machine learning techniques to classify individuals as diabetic or non-diabetic using basic health indicators and clinical symptoms.

## Dataset
The dataset contains 10 records with features such as age, blood sugar level, fatigue, excessive thirst, frequent urination, blurred vision, and weight loss.

## Files
- `diabetes_dataset.csv` – Sample symptom-based dataset
- `diabetes_detection.ipynb` – Jupyter Notebook with model training, evaluation, and visualization

## Tools Used
- Python
- Pandas
- Matplotlib
- Scikit-learn
